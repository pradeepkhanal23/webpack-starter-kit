## Webpack Starter Kit

Frontend environment for building Javascript applications.

Includes following features

- Babel Loader
- Sass Loader With Dart Sass
- HTMLWebpackPlugin
- WebpackDevServer
- Asset Resource Loader
- Source Maps
- Caching

## Usage

Run Dev Server (Port 3000)

```bash
npm run dev
```

Build for production

```bash
npm run build
```
